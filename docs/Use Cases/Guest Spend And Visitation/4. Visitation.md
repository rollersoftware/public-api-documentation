# Visitation

Each *BookingSignedWaiver* can be considered a visit.

The [**Tickets endpoint**](https://roller.stoplight.io/docs/roller-api/reference/Data%20API/DataAPI.yaml/paths/~1data~1tickets/get) response includes *ProductId* which can be used to quantify the number of visits for a specific product if required. This can be achieved by associating the *BookingSignedWaivers* to Tickets with a particular *ProductId*.

The quantity of bookings a customer has as the booking owner can be used, however, frequently the booking owner may not actually be a participant (ticket holder) and therefore would not be associated with a *BookingSignedWaiver*.
