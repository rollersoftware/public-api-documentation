# Webhooks Overview

The Webhooks API allows you to subscribe to events in ROLLER and receive a HTTP request to an endpoint you configure, instead of making an API call to ROLLER when an event occurs.

Webhooks are ideal for integrations which require data in real time, generally for operational purposes within your venue. You can configure from a range of subscribed events and payloads. 

See the sidebar for more information.

> For integrations which do not require real time data (such as a reporting integration) the [**ROLLER Data API**](overview.md) is better suited.
