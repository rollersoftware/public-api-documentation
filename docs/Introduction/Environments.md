> #### **Live Base URL**
> The base URL for Live is https://api.roller.app 

## Testing on the Playground Environment

To ensure the smooth implementation and functionality of new integrations, ROLLER recommends utilizing the Playground environment for initial testing rather than the Live environment.

<!-- theme: success -->
> #### **Playground Base URL**
> The base URL for Playground is https://api.play.roller.app 

### Benefits of using Playground

- **Parity**: Playground runs on the same version as the Live environment so is an accurate sandbox representation of behaviour
- **Scope Safety**: Changes and tests carried out in Playground do not affect your live data, minimizing the risk of disruptions to your operations. This enables isolated debugging so issues can be identified and resolved without affecting real-time transactions or customer interactions.

### Accessing the Playground Environment
To access Playground:

1. Log in to ROLLER Venue Manager
2. Select the Your Account icon at the bottom of the left-side menu.
3. Select "Switch to Playground venue"
4. You will be automatically redirected to your Playground account

>Learn more about Playground in our [**help center**](https://support.roller.software/hc/en-us/articles/360000298275-ROLLER-Playground).

<!-- theme: warning -->

> Please note: While using Playground to generate and host API keys currently in use, do not click the "**Update Playground**" button in Venue Manager - This will irreversibly reset the Playground instance and you will lose access to those keys.
