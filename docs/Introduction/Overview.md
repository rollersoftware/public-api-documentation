# Overview

The ROLLER Data API enables secure, structured access to your ROLLER data for integration with enterprise data warehouses, analytics platforms, and cloud storage solutions. This supports use cases like business intelligence, custom reporting, and data synchronization.


> - The Data API does not provide the ability to write data to ROLLER and should not be used for real-time data use cases as it does not allow for querying specific records and returns paginated data for a time period only 
> - It is optimised for the periodic export of records on a daily frequency
> - The available data is returned based on the modified date of the record, allowing for both new and adjusted records to be returned in the same call each day and updated in the external database
