# Getting API Access

> **If you're an existing ROLLER customer without the API access Add On:**
>
> Please reach out to your Client Success Manager or contact [**ROLLER Support**](https://support.roller.software/hc/en-us/articles/115001690793-Contacting-ROLLER-Support-?_ga=2.5806771.109159835.1679807311-440293524.1665358758&_gl=1*5d2are*_gcl_au*MTcxOTUwMTA0Ny4xNzQyOTY3MTU1*_ga*MTI1ODgzMjcyNC4xNzQyMjgwNTI1*_ga_QFK5BRXC35*MTc0MzM4MTY1Ny4xOS4xLjE3NDMzODE4MTkuMjQuMC4w)
> 
> **If you're not currently a ROLLER customer:**
>
> Please get in touch with us via our [**Contact Page**](https://www.roller.software/get-started/)

### **Steps to generate a ROLLER API key**
1. Login to [**Venue Manager**](https://manage.roller.app/)
2. Navigate to **Settings > Integrations > API keys**
3. Click the **Create client key** button.
4. Enter a description into _Client key name_ which best describes the consumer app or purpose that the key will be used for, for example `External Data Warehouse`.
5. Click **Create**.
