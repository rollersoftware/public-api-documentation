# Third-Party Checkout Overview

The ROLLER Rest API allows you to build your own online checkout outside of ROLLER and push the resulting bookings and payment records into ROLLER.

- Available products and capacities for specific dates can be called, and capacity can be reserved in advance of creating a booking.
- Bookings can then be created along with associated payments and customer records.
- Those bookings can be edited, cancelled, or deleted.
- Payments can be processed using the [**ROLLER Payments API**](https://docs.roller.app/docs/roller-payments/e4x9qrxpg5iwd).
